Comment ça marche

Simple comme appuyer sur une touche.

1

Maintenez la touche

Appuyez et maintenez votre hotkey (F9 par défaut). Un beep signale le début.

→
2

Parlez

Dictez ce que vous voulez. L'icône tray passe au rouge. Whisper écoute localement.

→
3

Relâchez

Un beep signale la fin. Whisper transcrit en 1-5s. Le texte s'injecte directement.

API HTTP Locale

Contrôlez Dictum depuis n'importe quelle app sur votre machine.

Endpoints disponibles

GET/statusÉtat complet en JSON
GET/historyHistorique des dictées
GET/configConfiguration active
POST/injectInjecter du texte
POST/pauseToggle pause
POST/reformulateReformuler via Ollama
POST/reloadRecharger la config
# Statut de Dictum
curl http://localhost:44880/status

# Réponse :
{
  "recording": false,
  "transcribing": false,
  "paused": false,
  "session_count": 12,
  "hotkey": "F9",
  "model": "ggml-medium.bin",
  "language": "fr",
  "version": "1.7.0"
}
# Injecter du texte à la position curseur
curl -X POST http://localhost:44880/inject \
  -d "Bonjour le monde"

# Réponse :
{ "ok": true }
# Reformuler via Ollama local
curl -X POST http://localhost:44880/reformulate \
  -H "Content-Type: application/json" \
  -d '{"text":"salut ca va","style":"email"}'

# Réponse :
{
  "result": "Bonjour,\n\nJ'espère que vous allez bien.\n\nCordialement"
}
